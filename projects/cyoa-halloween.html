CYOA Halloween Project

My class was assigned this project to help us express our creativity through prior knowledge about Github. It helped me practice using command lines, Git, and merging when collaborating with other people in one repo.

Something that was diffcult about this project is when all of my group members try to push their changes to the project. Each member in my group worked on different folders and files and it was frustrating to not get the changes we wanted just because someone else push theirs successfully. We also had to deal with merge conflict, which wasn't an easy thing to fix since everyone was trying to fix it all at once. I overcame this project by communicating with my group and checking in to see if they made any changes to the project so that I can pull theirs in and push my changes as well. I also had to use skills that I learned in this unit to help fix problems not only for me, but for my group members as well.

Something that I would have done differently if I had more time was to include images. Visuals can convey audience to be engaged of what the designer is showcasing. Although the context in my project was descriptive, I feel like adding images would add a spark in my story and make it more interesting and fun.
