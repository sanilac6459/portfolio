SEP11 Freedom Project: Keysanity Typing Game

For this year long project, my class used Javascript which was the course of SEP11 and a tool we choose to create anything we want. So, me and my group decided to make Keysanity which a speed typing game.

For our creation, we used JS and our tool, p5js, to make a fun and interactive game for users to play and it's a great way for them to be efficient with their typing skills which can be put on their resumes.

The most challenging part of making our project was deciding what to make. Since we had to start deciding at the beginning of the year, my group and I was unsure how much we need to learn to make our creation since we haven't learn a lot about Javascript at that time. Another challenge was deciding on a tool to use because when my group and I were tinkering, we followed tutorials and documentations but still found it confusing to process. Personally, I felt like my group and I was trying to make something complex but didn't have a lot of resource to help us make it. We also had to deal with a lot of merge conflict. Since all of us worked on our project at the same time, it was causing our code to break many times as we tried to push our changes.

My biggest takeaway from this project was time management. Since we only worked on it for a short period of time in class on Monday, my group and I didn't really get a lot of done and it was hard to work on it at home due to work from other classes and procrastination. We mostly finished our project during spring break which was a lot to do to get our game to work. Another takeaway I had was understanding the code my group wrote. Although we had time to tinker each part we need in our game, it was still a lot to process of what each chunk of code does. We decided to comment the ones that we're probably going to forget when we look back at it later.
