Halloween Hackathon: GIMME CANDY!!!

As part of the Daedalus Computer Science Honors at Hunter College, we participated in the Halloween Hackathon where we can make anything we want related to halloween. My group and I decided to make a trick or treat game called GIMME CANDY!!!

For our creation, we used Unity which is a game engine platform to created interactive and real 2D and 3D content.

How our game works is the player starts off with 20 candies and has to go as many houses as they can to receive more within 5 minutes. Along the way, there are enemies that will try to steal the candy so the player will have to be careful and avoid encountering them. The game is over when the timer is up or when the player lose all their candy.

The most challenging part of this project was using Unity. For me, it wasn't an easy tool to learn. During the process of making this game, I was lost, even after looking at tutorials. For the most part, only one of my group member used Unity before so he helped me and my other group members get our parts done. Another challenge I faced was my code editor. For some reason, my code editor wasn't set as the default code editor for Unity and when I was looking up how to set it, my computer didn't have a category in Unity setting to set it my code editor as the default. In the end, I had to focus more on the front-end like designing the menu and game over scene to make it look aesthetic.
